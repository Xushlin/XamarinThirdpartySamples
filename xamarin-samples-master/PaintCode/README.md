PaintCode Demo
==============

PaintCode (www.paintcodeapp.com) now supports C# code generation for Xamarin MonoTouch.

This sample app shows some PaintCode-generated elements:

* Calendar badge

* Blue button 

* Glossy button

* Random line-art

See also http://blog.xamarin.com/
